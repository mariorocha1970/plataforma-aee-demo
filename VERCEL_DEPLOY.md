# Publicação no Vercel

1. Importe este repositório como um novo projeto Vercel.
2. Confirme que o framework detetado é **Next.js**.
3. Em **Environment Variables**, crie:
   - `DEMO_PASSWORD`: palavra-passe que será partilhada com os visitantes.
   - `DEMO_SESSION_TOKEN`: token longo e aleatório, não partilhado.
4. Publique o projeto.
5. Se o endereço pretendido estiver disponível, altere o nome do projeto para `plataforma-aee` e obtenha `plataforma-aee.vercel.app`.

O repositório não contém os valores reais dos segredos.
